# フランス革命シミュレーター 🇫🇷

SSD Engine を用いた1789-1794年フランス革命の歴史的再現

## 📜 概要

フランス革命は、近代史上最も劇的な社会変革の一つです。このシミュレーターは、構造主観力学（SSD）理論を用いて、革命の勃発から恐怖政治、そしてテルミドール反動までのプロセスを再現します。

## 🎭 社会階層

シミュレーションには5つの社会階層が登場します：

### 👑 貴族（Nobility） - 3エージェント
- **特徴**: 極めて高い特権意識、変化への強い抵抗
- **初期状態**: 
  - κ_UPPER = 4.0（王権神授説への強い信念）
  - κ_CORE = 3.5（貴族の名誉）
  - E_BASE = 1.0（比較的安定）

### ⛪ 聖職者（Clergy） - 3エージェント
- **特徴**: 宗教的権威、保守的
- **初期状態**:
  - κ_UPPER = 3.5（宗教的信念）
  - κ_CORE = 3.0（教会の権威）

### 💼 ブルジョワジー（Bourgeoisie） - 5エージェント
- **特徴**: 啓蒙思想、立憲君主制志向
- **初期状態**:
  - κ_UPPER = 2.5（自由主義思想）
  - E_UPPER = 2.0（政治的野心）
  - E_BASE = 2.0（体制への不満）

### 🔥 サンキュロット（Sans-culottes） - 8エージェント
- **特徴**: 急進的都市労働者、直接行動志向
- **初期状態**:
  - κ_UPPER = 1.5（急進共和主義）
  - E_BASE = 4.0（高い怒りと飢え）
  - E_UPPER = 3.0（革命的情熱）

### 🌾 農民（Peasants） - 12エージェント
- **特徴**: 保守的だが経済的圧力が高い
- **初期状態**:
  - E_BASE = 5.0（飢餓と重税）
  - E_PHYSICAL = 3.0（肉体的疲労）

## 📅 歴史的フェーズ

### 1. 革命前夜（1788-1789） - Step 0-59
**状況**:
- 財政危機による国家破産の危機
- 連続する不作による食糧不足
- 三部会召集の決定

**圧力**:
- 貴族: 特権への不安
- ブルジョワジー: 政治的野心の高まり
- サンキュロット・農民: 飢えと怒り

### 2. 革命の勃発（1789-1792） - Step 60-119
**主要イベント**:
- 🏰 バスティーユ襲撃（Step 60）
- 封建制廃止
- 人権宣言
- 立憲君主制の試み

**圧力**:
- 貴族: 恐怖とパニック、特権の崩壊
- ブルジョワジー: 立憲君主制への希望
- サンキュロット: 革命の興奮、共和主義の高揚

### 3. 急進化・恐怖政治（1792-1794） - Step 120-219
**主要イベント**:
- 👑 王政廃止宣言（Step 120）
- ⚔️ 恐怖政治開始（Step 150）
- ジャコバン独裁
- 大量処刑

**メカニズム**:
- 処刑リスク: 貴族 1%/step、ブルジョワジー 0.5%/step
- 恐怖レベルの増加
- 革命熱の最高潮

### 4. テルミドール反動（1794） - Step 220-299
**主要イベント**:
- 🔻 テルミドール9日のクーデター（Step 220）
- ロベスピエール失脚
- 恐怖政治の終焉

**変化**:
- サンキュロット: 失望、理想の崩壊
- ブルジョワジー: 安堵、新秩序の構築
- 革命熱の減衰

## 🔬 SSD理論による分析

### E/κダイナミクス

**E（未処理圧）の役割**:
- BASE層: 飢餓、恐怖、怒りの蓄積
- CORE層: 規範との葛藤（従来の価値観 vs 新しい思想）
- UPPER層: イデオロギー的圧力（王権 vs 共和制）

**κ（慣性）の役割**:
- 貴族の高いκ → 変化への強い抵抗 → 革命への反発
- サンキュロットの低いκ → 急進化しやすい → 恐怖政治の推進力

### 社会的カップリング

**エネルギー伝播（ζ）**:
- ζ_BASE = 0.18: 恐怖と怒りが急速に伝染
- ζ_UPPER = 0.08: 革命思想の拡散

**対立の増幅（ω）**:
- ω_BASE = -0.12: 階級間の対立が感情を増幅

### 階級間関係

```
貴族 ←→ サンキュロット/農民: -0.9（強い対立）
ブルジョワジー ←→ 貴族: -0.6（中程度の対立）
ブルジョワジー ←→ サンキュロット: 0.4 → -0.3（協力から対立へ）
```

## 📊 分析指標

シミュレーションは以下の指標を追跡します:

1. **貴族の権力**: κ_UPPER の平均
2. **革命熱**: サンキュロットの E_UPPER 平均
3. **社会的緊張**: 全体の E_BASE 平均
4. **恐怖レベル**: 処刑リスクと集団的恐怖
5. **処刑者数**: 累積処刑者数
6. **階級別エネルギー**: 各階級のストレスレベル
7. **階級別イデオロギー**: 各階級の信念の強さ

## 🎯 期待される結果

### 定性的傾向

1. **貴族の権力衰退**
   - 初期: κ_UPPER ≈ 4.0
   - 最終: κ_UPPER < 1.0
   - 急激な崩壊曲線

2. **革命熱の高揚と減衰**
   - 革命前夜: E_UPPER ≈ 3.0
   - 恐怖政治: E_UPPER > 6.0（ピーク）
   - テルミドール: E_UPPER < 2.0

3. **社会的緊張の推移**
   - 革命前夜: 高い緊張（E_BASE ≈ 4.0）
   - 革命の勃発: 一時的緩和
   - 恐怖政治: 極度の緊張（E_BASE > 6.0）
   - 反動期: 緩やかな低下

4. **処刑者数**
   - 総数: 5-10人（元の31人中）
   - 大半は貴族とジロンド派ブルジョワジー

### 創発的パターン

- **分極化**: ブルジョワジーとサンキュロットの対立深化
- **恐怖の連鎖**: 処刑が新たな恐怖を生む正のフィードバック
- **疲弊**: 長期化する革命による全階級の消耗
- **反動の必然性**: 極度の緊張からの揺り戻し

## 💡 歴史的洞察

このシミュレーションから得られる洞察:

### 1. 革命の必然性
- 高いE（不満）+ 低いκ_min（柔軟性の欠如）= 構造的跳躍（革命）
- 貴族の妥協不可能な態度（高κ_UPPER）が緩やかな改革を不可能に

### 2. 急進化のメカニズム
- 初期の協力（ブルジョワジー ↔ サンキュロット）
- 対立の深化（階級間のE伝播と増幅）
- サンキュロットの低κ → 急進化への易しさ

### 3. 恐怖政治の論理
- 外敵の脅威 → E_BASEの増大
- 「革命の防衛」という正当化 → E_UPPERの維持
- 処刑による恐怖の拡散 → さらなる処刑

### 4. 反動の必然性
- 人間の持続可能な緊張レベルの限界
- E_BASEの減衰（β_base）
- 疲弊による革命熱の低下

このシミュレーターは以下を示します:

1. **ミクロ-マクロリンク**
   - 個人の内部力学（E/κ）→ 集団レベルの革命

2. **創発的秩序**
