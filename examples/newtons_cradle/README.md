## SSD理論の物理適用

### 三層構造の物理的解釈

**BASE層（位置・運動量）**
- 基本的な力学状態
- 位置と速度の保存

**CORE層（衝突相互作用）**
- 球間の衝突力
- エネルギー伝達

**UPPER層（システム全体）**
- 全体の運動パターン
- エネルギー保存則

### 意味圧の物理的源泉

1. **衝突圧**: 球同士の接触による力
2. **復元圧**: 弾性力による反発
3. **伝達圧**: 運動量・エネルギーの伝播

## 物理パラメータ

各実装で設定可能:

- **球の数**: 5個（標準）
- **質量**: 均一 or 可変
- **反発係数**: 0.95（ほぼ弾性衝突）
- **時間刻み**: nano版は極小ステップ

## 理論的特徴

### E/κダイナミクスの物理版

- **E（圧）**: 衝突による瞬間的な力
- **κ（慣性）**: 質量による変化への抵抗
- **β（減衰）**: エネルギー損失率

### 創発的パターン

外部制御なしで以下が創発:

1. **波動伝播**: 運動量の順次伝達
2. **対称振動**: 左右対称な運動パターン
3. **エネルギー保存**: 系全体での保存則
