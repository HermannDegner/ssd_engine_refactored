## 🎯 理論的基盤

### E/κダイナミクス

SSDエンジンは、人間の意思決定を以下の内部力学でモデル化します:

- **E (未処理圧)**: 状況的・一時的な圧力
  - 処理しきれなかった圧力が蓄積
  - 閾値を超えると構造的跳躍（意思決定の転換）
  
- **κ (慣性)**: 確立された価値・変化への抵抗
  - 経験により強化される経路の「使いやすさ」
  - 高いκは安定性をもたらすが、柔軟性を低下させる

### 三層構造

人間の意思決定は3つの層で行われます:

1. **BASE層** - 本能・感情（恐怖、怒り、欲求）
2. **CORE層** - 規範・自我（社会的規範、自尊心）
3. **UPPER層** - 戦略・理念（イデオロギー、長期計画）

### 社会的カップリング

エージェント間の相互作用:

- **エネルギー伝播** (ζ係数): 感情や圧力の伝染
- **κ伝播** (ξ係数): 価値観や規範の伝播
- **競合抑制** (ω係数): 対立関係での相互増幅

## 🚀 カスタマイズ

### エージェント数の変更

```python
# social_analysis_demo.py
scenario = OpinionPolarizationScenario(num_agents=10)  # デフォルト: 6
```

### シミュレーションステップ数の変更

```python
scenario.run_simulation(num_steps=200)  # デフォルト: 100
```

### カップリングパラメータの調整

```python
coupling_params = SocialCouplingParams()
coupling_params.zeta_base = 0.20  # BASE層エネルギー伝播を強化
coupling_params.xi_core = 0.12    # CORE層κ伝播を強化
```

### 関係性マトリクスのカスタマイズ

```python
# 完全協力関係
relationships = RelationshipMatrix.create_cooperative(num_agents)

# ランダム関係（協力バイアス付き）
relationships = RelationshipMatrix.create_random(num_agents, cooperation_bias=0.5)

# カスタム関係性
relation_matrix = np.zeros((num_agents, num_agents))
relation_matrix[0, 1] = 0.9  # Agent 0 → Agent 1: 強い協力
relation_matrix[1, 0] = -0.5  # Agent 1 → Agent 0: 中程度の競争
relationships = RelationshipMatrix(matrix=relation_matrix)
```
