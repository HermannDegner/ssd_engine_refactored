## Log版エンジンの特徴と利点

### 🚀 Log-Alignment技術
```
変換式: p̂ = sign(p)·log(1+α_t|p|)/log(b)
適応ゲイン: α_t (EMA更新、信号強度に応じて自動調整)
```
- **大信号適応**: 信号が1000倍になってもα_tが1.0→0.01に自動調整
- **数値安定性**: 極端な値でも安定した処理
- **圧縮効果**: 大信号を効率的に圧縮（圧縮率1740倍まで対応）

### 🎲 確率的跳躍システム
```
跳躍確率: P(leap) = 1/(1 + exp(-(E-Θ)/T))
温度パラメータ: T (0.0～15.0で自然な確率分布)
```
- **自然な非線形性**: 温度による確率的な状態遷移
- **予測可能性**: パラメータ制御による行動調整

### 📊 包括的診断システム
```
記録項目:
- theta_dynamic: 動的閾値
- power: エネルギー密度  
- dominant_layer: 支配層
- pressure_hat_norm: 変換後圧力ノルム
```
- **システム状態監視**: リアルタイム診断情報
- **デバッグ支援**: 詳細な内部状態追跡
